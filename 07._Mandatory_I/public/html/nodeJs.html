1. Basics

This code example covers fundamental concepts related to variables, strings, and numbers. It consists of two exercises that demonstrate the use of these concepts in JavaScript

In this exercise, you are introduced to two variables, firstName and lastName, which hold string values. The goal is to display a message in the console that combines these variables to form a sentence. The code showcases two methods of achieving this: using template literals and traditional string concatenation. Running this code will print your first and last name in two different way

                    
const firstName = "Anders";
const lastName = "Latif";

console.log(`My first name is ${firstName} and my last name is ${lastName}`)
console.log("My first name is", firstName,"and my last name is", lastName)

The second exercise provides two variables, year and number. Your task is to add the value of number to the value of year while adhering to the constraint that you cannot modify lines 1 or 2. To address this challenge, the code demonstrates three distinct methods of adding a number to a string: parseInt, Number, and unary + operator. The results are printed to the console.

          
const year = "2022";
const number = 1;

// Add the year plus the number
// The result should be 2023
// You cannot touch line 1 or 2

const newYear = parseInt(year) + number;
const newYear2 = Number(year) + number;
const newYear3 = +year + number;

console.log(newYear);
console.log(newYear2);
console.log(newYear3);

2. Numbers and Strings

In this exercise, two variables, numberOne and numberTwo, hold string representations of decimal numbers. The objective is to add these numbers together and display the result as a float. The code uses the parseFloat function to convert the string values to floats and then performs the addition.

              
const numberOne = "1.10";
const numberTwo = "2.30";

// add those two numbers and show the result
// you cannot touch line 1 neither line 2

const resultOne = parseFloat(numberOne) + parseFloat(numberTwo)

console.log(resultOne);                
            
        
              
const numberOne = "1.10";
const numberTwo = "2.30";

const resultTwo = parseFloat(numberOne) + parseFloat(numberTwo);

console.log(resultTwo.toFixed(2));

Similar to the previous exercise, this exercise involves adding two numbers represented as strings. However, it adds the extra step of formatting the result to display it with precisely two decimal places using the toFixed method.

In this exercise, three numeric variables (one, two, and three) are provided, and the goal is to calculate their average with five decimal places. The code performs the necessary arithmetic operations and utilizes the toFixed method to control the decimal precision.

              
const one = 10;
const two = 45;
const three = 98;

// Show in the console the avg. with 5 decimals

const average = (one+two+three) / 3;

console.log(average.toFixed(5));

In the final exercise, a string variable fact contains the text "You are learning javascript!" The objective is to capitalize the lowercase "j" in "javascript." The code utilizes the replace method to achieve this text manipulation.

              
const fact = "You are learning javascript!";

// capitalize the J in Javascript

const factDone = fact.replace('j','J');

console.log(factDone);

3. Objects

This code snippet focuses on working with JavaScript objects. Objects are fundamental data structures that allow you to store and manipulate data in a structured manner. The following exercises provide hands-on experience with working with objects, a crucial concept in JavaScript for organizing and managing data.

In this exercise, you are introduced to an object named myObj that contains a single key-value pair. The objective is to retrieve and log the value associated with the key "message." This exercise illustrates how to access values from an object using dot notation.

                
const myObj = { message: "Hello, earthling! I bring peace." };

// Log the message 

console.log(myObj.message);            
            
        
              
const tObj = { name:"Thomas", age: 31}

Here, you are tasked with creating your own object named tObj that holds your name and age as properties. This exercise demonstrates how to define and initialize objects with custom key-value pairs.

In this exercise, an empty object named stackOverflow is provided, and your goal is to add a property named "isAllowed" with a value of true to it. The code showcases how to add properties to an existing object.

                
const stackOverflow = {};

stackOverflow.isAllowed = true;

// make a rule called isAllowed and let the value be true

console.log(stackOverflow.isAllowed);              
            
        
              
const thisSong = { description: "The best song in the world." };

// remove the property "description" 
// and add a property called "about" that should say "Just a tribute." 

delete thisSong.description;

thisSong.about = "Just a tribute."

console.log(thisSong.about);

The final exercise involves an object called thisSong with a "description" property. The task is to remove the "description" property and add a new property named "about" with the value "Just a tribute." This exercise demonstrates property deletion and modification in JavaScript objects.
