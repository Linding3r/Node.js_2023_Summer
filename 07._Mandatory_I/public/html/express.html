Express

Express is a popular and widely used web application framework for Node.js, a runtime environment for executing JavaScript code on the server side. Express is designed to simplify the process of building web applications and APIs in Node.js by providing a set of powerful and flexible features and tools.

Setup

Open your terminal and navigate to the directory where you want to create your Express project. Use the following command to initialize a new Node.js project which create a package.json file:

    
npm init -y

Next, you'll need to install Express.js as a project dependency. Run the following command:

         
npm install express
        
    

        
            
import express from 'express';

const app = express();
            
const PORT = 8080;
            
app.listen(PORT, () => {
    console.log('Server running on port', PORT);
});

Application File

This code serves as the foundation for building web applications and is the starting point for web development with Express.js. This is a basic setup for creating a web server using the Express framework in JavaScript. It starts by importing the Express library and then initializes an Express application instance. The server is configured to listen on port 8080, which is a common choice for web applications. When the server starts, it prints a message to the console, indicating that it's running on port 8080.

Routing

Express allows you to define routes that map HTTP methods (e.g., GET, POST, PUT, DELETE) to specific URL patterns. This makes it easy to handle different types of requests and define how the application responds to them.

Here are some examples of routes using different HTTP methods and URL patterns in Express:

Start Application

        
node app.js

To start your Express application, run the following command in your terminal. 

        
nodemon app.js

If you install the nodemon package, you can use the nodemon command instead of node to start your application. Nodemon will automatically restart your application when you make changes to your code, which is useful for development.
