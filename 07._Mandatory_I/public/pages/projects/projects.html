Mountain API


This project was developed using Node.js and Express.js, creating a REST API. The API empowers users to perform essential CRUD (Create, Read, Update, Delete) operations on a database containing mountain data.


Time To Deploy


For our inaugural freestyle project, we centered our theme around "time." Our goal was to create a website that offers intriguing facts about various inventions throughout history. Additionally, we implemented an interactive game challenging users to guess the invention years.

  • Invention Insights: The website provides users with educational content about historical inventions, highlighting their significance and evolution over time.
  • Guess-the-Year Game: Users engage in an entertaining game that tests their knowledge of invention timelines, adding an element of fun and learning.
