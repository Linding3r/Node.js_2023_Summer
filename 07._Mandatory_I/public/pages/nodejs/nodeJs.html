2. Numbers and Strings

In this exercise, two variables, numberOne and numberTwo, hold string representations of decimal numbers. The objective is to add these numbers together and display the result as a float. The code uses the parseFloat function to convert the string values to floats and then performs the addition. 

              
const numberOne = "1.10";
const numberTwo = "2.30";

// add those two numbers and show the result
// you cannot touch line 1 neither line 2

const resultOne = parseFloat(numberOne) + parseFloat(numberTwo)

console.log(resultOne);                
            
        
              
const numberOne = "1.10";
const numberTwo = "2.30";

const resultTwo = parseFloat(numberOne) + parseFloat(numberTwo);

console.log(resultTwo.toFixed(2));

Similar to the previous exercise, this exercise involves adding two numbers represented as strings. However, it adds the extra step of formatting the result to display it with precisely two decimal places using the toFixed method.

In this exercise, three numeric variables (one, two, and three) are provided, and the goal is to calculate their average with five decimal places. The code performs the necessary arithmetic operations and utilizes the toFixed method to control the decimal precision. 

              
const one = 10;
const two = 45;
const three = 98;

// Show in the console the avg. with 5 decimals

const average = (one+two+three) / 3;

console.log(average.toFixed(5));

console.log(myObj.message);
